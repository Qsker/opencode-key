# opencode-key

Interactive switcher for opencode API keys. Cross-platform (macOS/Linux/Windows).

Auto-detects all providers from `auth.json`.

## Install

```bash
curl -sS https://raw.githubusercontent.com/Qsker/opencode-key/main/opencode-key > ~/.local/bin/opencode-key
chmod +x ~/.local/bin/opencode-key
```

## Usage

```
opencode-key            # interactive menu
opencode-key list      # list all keys
opencode-key current   # show current keys
opencode-key add       # add a key
```
